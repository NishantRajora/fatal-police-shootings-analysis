# Fatal Police Shootings Data Analysis

This project analyzes and visualizes trends in fatal police shootings in the United States using the dataset from [Washington Post](https://github.com/washingtonpost/data-police-shootings).

## 📊 Features

- Gender and Race distribution
- Age distribution
- Weapon/Armed status trends
- Mental illness signs
- Time-series monthly analysis
- Top 10 states with most cases
- Interactive geographical map (Plotly)
- Correlation matrix
- Body camera usage comparison

## 📁 Files

- `fatal-police-shootings-analysis.ipynb`: Jupyter notebook with full code
- `fatal-police-shootings-data.csv`: Dataset used
- `/outputs`: Folder containing all output graphs and HTML map
